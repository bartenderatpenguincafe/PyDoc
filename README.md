# PyDock (Docklight-like, COM only)

## Сборка EXE
`build\build.bat` → `build\dist\PyDock\PyDock.exe` (portable one-folder).

## Настройки проекта (COM)
Порт, baud, биты данных, чётность, стоп-биты, контроль потока, LSB/MSB.

## Настройки приложения
Тема (light/dark), шрифт/размер, цвета RX/TX/SYS, компактный интерфейс.
Файл: `%APPDATA%\PyDock\settings.json`.
